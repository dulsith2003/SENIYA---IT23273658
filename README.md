# This is an automated playwright project that contains some tests for https://www.swifttranslator.com/

## How to install dependencies
1. npm install (Install all the dependencies need for the project)
2. npx playwright install --with-deps (Install playwright browsers with dependencies)

## Run tests
npx playwright test (Run all tests)

npx playwright test -g "test ID" (Run a specific test. Use the test ID of the test you want to run in the "Test ID" section)

## Some positive tests may fail during the execution of all tests. In that case, run the test individually.
